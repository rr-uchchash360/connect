# Connect — Personal Hub

[![Live Site](https://img.shields.io/badge/Live-Connect-blue)](https://rr-uchchash360.github.io/connect/)

Modern link-in-bio page with dark/light mode, glassmorphism design, and centralized contact/social links.

## Features
- 🌓 Dark/Light mode (auto + manual)
- 📱 Fully responsive
- ⚡ Vue 3 + vanilla CSS
- 🔗 Phone, Telegram, WhatsApp, Messenger, Email
- 📱 Facebook, Instagram, X, LinkedIn

## Tech Stack
- Vue.js 3
- HTML5/CSS3
- GitHub Pages

## Local Setup
```bash
git clone https://github.com/rr-uchchash360/rr-uchchash360.github.io.git
cd rr-uchchash360.github.io/connect
python -m http.server 8000
```

## Customize
Edit `ProfileModel`, `ContactModel`, or `SocialModel` in `index.html`
